## Usage
 Step 1.Add the JitPack repository to your build file 
 ```kotlin
allprojects {
	repositories {
		maven { url 'https://jitpack.io' }
	}
}
  ```
 Step 2. Add the dependency
 ```kotlin
dependencies {
    implementation 'com.github.imanih20:persian-android-datepicker:<LATEST-VERSION>'
}
  ```
